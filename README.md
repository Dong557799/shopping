# 电子购物商城

## 项目简介
基于 Vue3 + Vite 搭建的移动端生鲜电商平台，完整实现生鲜购物全业务流程，适配手机端交互。包含商品浏览、分类筛选、购物车、订单结算、地址管理、个人中心等全套电商功能，代码结构清晰，易于二次开发与对接后端真实接口。

## 技术栈
- 构建工具：Vite
- 前端框架：Vue3
- 路由管理：Vue Router
- 全局状态：Pinia
- UI组件库：Element Plus
- 网络请求：Axios
- 样式处理：SCSS

## 功能模块
### 1. 首页
轮播广告、商品分类导航、爆款推荐、限时活动，支持下拉刷新、上拉加载更多。

### 2. 商品分类
左侧大类导航，右侧对应商品列表，生鲜、果蔬、肉禽、零食多品类快速筛选。

### 3. 商品详情
大图预览、规格选择、库存展示、加入购物车、立即购买、商品图文、用户评价。

### 4. 购物车
商品多选、数量增减、单品删除、全选/反选、实时总价计算、跳转结算页。

### 5. 订单结算
收货地址选择/新增/编辑、优惠券抵扣、商品清单预览、计算实付、提交订单。

### 6. 个人中心
用户信息展示、全部订单（待付款/待发货/已完成）、优惠券、地址管理、客服入口。

### 7. 地址管理
新增、编辑、删除收货地址，支持设置默认地址。
